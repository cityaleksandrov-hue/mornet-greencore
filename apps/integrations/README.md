# Integrations

Сервисы интеграции с внешними системами (банки, биржи, compliance).

## Сервисы

### 1. Bank Gateway
Интеграция с банками-партнёрами для fiat on/off-ramp.

### 2. Exchange Router
Интеграция с криптовалютными биржами.

### 3. GC-Guard Adapter
Интеграция с GC-Guard для KYC/AML/compliance.
